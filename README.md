# 爱高创客学员能力测评 H5

基于 React + Vite 的 H5 原型，支持学员信息录入、能力评分、雷达图展示、报告文字自动生成、历史测评记录管理，以及浏览器 localStorage 本地保存。

## 本地启动

```bash
npm install
npm run dev
```

启动后在浏览器打开终端显示的本地地址，通常是：

```text
http://localhost:5173
```

## 功能范围

- 录入学员姓名、年级、课程体系、课程类型、阶段时间、授课老师
- 项目资料库支持新增、编辑、删除项目，并按年级、课程体系、课程类型筛选
- 测评表单可按年级、课程体系、课程类型多选本阶段学习内容，并自动汇总学习内容、对应能力、阶段成果描述和家长报告描述
- 录入测评周期和可选老师备注，报告主要根据能力评分和历史对比自动生成阶段分析
- 根据课程体系自动切换能力维度，并进行 1-5 分评分
- 评分区提供通用评分参考标准，标准配置维护在 `src/constants/assessment.js`
- 历史记录会保存当次使用的能力维度，后续配置调整不影响旧记录
- 根据学员姓名自动查找历史记录，支持首次基线报告和阶段成长对比报告
- 有对比记录时，雷达图显示“上次测评”和“本次测评”两组数据
- 自动生成能力雷达图
- 自动生成综合评价、优势能力、待提升能力、下一阶段建议
- 保存测评记录到浏览器 localStorage，刷新后仍可查看
- 独立历史测评记录页面，支持按姓名搜索和删除记录
- 同一个学员支持保存多次测评
- 独立竞赛能力测评入口，支持机器人竞赛六维评分、行为选项自动评分、手动微调、报告生成和历史记录
- 点击历史记录查看完整家长版报告
- 家长版报告适合手机竖版截图
- 分享卡片版报告适合手机一屏截图，卡片区域使用 `#parent-share-card` 包裹，后续可接导出图片
- 支持将分享卡片单独导出为高清 PNG，按钮区域不会进入导出图片
- 支持复制适合发送给家长微信的文字版报告

## 代码结构

```text
src/
  components/          页面组件
  constants/           测评维度、课程、年级等配置
  services/            本地存储服务，后续可替换为接口请求
  utils/               报告生成和雷达图数据转换逻辑
  App.jsx              主页面和业务流程
  styles.css           全局样式
```

后续接入数据库、登录和教务管理时，可以优先替换 `src/services/storage.js`，并把 `records` 和 `projects` 数据流改为接口读取与提交。
